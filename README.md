# YAMA · Violin

> Keman pratiği ve ısınma — Android uygulaması
> Geigen-Übung & Aufwärmen — Android-App · Violin practice & warm-ups — Android app

**TR —** Keman çalışanlar için pratik ve ısınma uygulaması. Kotlin / Jetpack Compose ile geliştirilmiş bir **Android** uygulamasıdır; tarayıcıda açılan bir web sürümü yoktur.

**DE —** Eine **Android-App** zum Üben und Aufwärmen für Geiger:innen, entwickelt mit Kotlin / Jetpack Compose. Es gibt keine Web-Version.

**EN —** A practice-and-warm-up **Android app** for violinists, built with Kotlin / Jetpack Compose. There is no browser-based web version.

> Kurulum: Kaynak koddan Android Studio ile derlenir. (İleride bir APK sürümü eklenebilir.)
